# Book Searcher
---
![screen](../../screen.png)

### Описание
---
React-приложение по поиску книг с помощью [Google Books API](https://developers.google.com/books/docs/v1/using).
Адаптивная верстка присутствует. 
Цель проекта: изучить основы Redux Toolkit.

### Технологии
---
- ReactJS
- React Router
- Redux Toolkit
- CSS
- Google Books API
